# 从零开始的 Python 完全教程

> **目标读者**：编程零基础，想要真正「读懂代码」并能「用代码解决实际问题」的人。
>
> **核心理念**：不讲花活，只讲原理。每行代码你都能理解它在内存里干了什么。

---

## 📚 教程结构

本教程分为 **四个阶段**，每个阶段结束时你都能做出实际有用的东西：

| 阶段 | 主题 | 学完你能做什么 | 
|------|------|---------------|
| **第一阶段** | 程序是怎么跑起来的 | 理解变量、类型、控制流，写计算器和猜数字游戏 |
| **第二阶段** | 数据的世界 | 处理文件、整理 Excel 数据、生成报表 |
| **第三阶段** | 构建应用 | 写 Web API、爬取网页、自动化办公 |
| **第四阶段** | 工程思维 | 测试、调试、Git 版本控制、部署 |

## 🚀 怎么用这个教程

1. **按顺序读**。每一章都依赖前面的知识。
2. **不要复制粘贴代码**。亲手敲一遍，犯错了更好——你在学调试。
3. **改代码实验**。读完示例后，故意改动看看会发生什么。
4. **完成每个阶段的「实战项目」**。这是把知识变成技能的关键步骤。

## ⚡ 环境准备

在开始之前，你只需要：

1. **安装 Python**（推荐 3.10+）：[python.org](https://python.org)
2. **装一个编辑器**：推荐 [VS Code](https://code.visualstudio.com/)
3. **打开终端**，输入 `python --version` 确认安装成功
