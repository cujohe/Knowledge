# 第一章：程序是什么

## 1.1 从一个类比开始

你早上起床的流程：

```
1. 睁眼
2. 如果 闹钟在响 → 关掉闹钟
3. 去卫生间
4. 如果 热水器没开 → 打开热水器，等 5 分钟
5. 刷牙、洗脸
```

这就是一个 **程序**——一系列按顺序执行的指令，遇到条件时做判断。

**计算机程序的本质**：把你脑子里的「做事的步骤」翻译成计算机能理解的语言。计算机不会思考，它只会**严格按你写的步骤执行**。

> 💡 **核心认知**：写代码不是「让计算机变聪明」，而是「你自己要想清楚步骤」。计算机只是你的一个**极其听话但完全不会自己动脑子的执行者**。

---

## 1.2 计算机是怎么「理解」指令的？

这是一个层级的翻译过程：

```
你写的 Python 代码（人能读）
        ↓
Python 解释器翻译成「字节码」
        ↓
Python 虚拟机执行字节码
        ↓
操作系统把指令交给 CPU
        ↓
CPU 执行机器码（010101010111...）
```

你不需要记住这个全流程。关键理解一点：**Python 解释器是中间翻译官**。它把你写的 `print("hello")` 翻译成计算机能做的操作。

对比其他语言：

| 语言 | 翻译方式 | 特点 |
|------|---------|------|
| **Python** | 解释执行（边翻译边跑） | 开发快，运行慢一点 |
| **C / C++** | 编译执行（先全部翻译完再跑） | 开发慢，运行快 |
| **Java** | 先编译再解释（混合） | 折中方案 |

> 💡 Python 为什么适合初学者？因为你改了一行代码，直接就能跑，不需要等编译。**反馈越快，学得越快。**

## 1.3 你的第一个程序

打开 VS Code，新建文件 `hello.py`，输入：

```python
print("Hello, 世界!")
```

打开终端，运行：

```bash
python hello.py
```

输出：
```
Hello, 世界!
```

---

### 这一行代码底层发生了什么？

当你输入 `print("Hello, 世界!")` 然后敲回车时：

1. **词法分析**：Python 解释器把这一行切成 token（词法单元）：
   - `print` → 一个标识符（函数名）
   - `(` → 左括号
   - `"Hello, 世界!"` → 一个字符串
   - `)` → 右括号

2. **语法分析**：把 token 组成抽象语法树（AST）：
   ```
   函数调用
   ├── 函数名: print
   └── 参数: "Hello, 世界!"
   ```

3. **执行**：Python 发现 `print` 是一个内置函数，调用底层 C 代码，把 `"Hello, 世界!"` 这个字符串通过系统调用发送到终端（标准输出），终端把文字显示在屏幕上。

> 🧠 **记住这个模式**：你的代码 → Token → AST → 执行。后续学习任何新语法，这个模式都不会变。

## 1.4 Python 的交互模式：一个更好的实验场

除了把代码写进文件，你还可以直接在终端输入 `python`，进入交互模式：

```python
>>> 2 + 3
5
>>> print("你好")
你好
>>> 10 * 5
50
```

**每输入一行，Python 立刻执行并显示结果。** 这是学 Python 最强大的工具——你可以随时试，错了也无所谓。

当你在后续章节看到一段代码时，**先别信我**——打开交互模式，自己敲一遍，改数字、改文字，看会发生什么。

---

## 1.5 代码执行的两条铁律

> **铁律一：从上到下，逐行执行。**
>
> **铁律二：每行执行完，再执行下一行。**

```python
print("第一行")
print("第二行")
print("第三行")
```

输出：
```
第一行
第二行
第三行
```

这听起来简单到可笑——但**大部分 bug 都是因为忘了这两条铁律**。比如你以为某行代码已经执行了，但它其实被跳过了（因为有条件判断），或者你以为一个变量已经有值了，但赋值在它后面。

## 1.6 注释：写给人类看的代码

```python
# 这是一个注释。Python 会完全忽略它。
print("这行会执行")   # 这也是注释
# print("这行不会执行")
```

注释的作用是**解释你为什么这么写**，而不是重复代码做了什么。

❌ 糟糕的注释：
```python
# 把 a 加 b 赋值给 c
c = a + b
```

✅ 有用的注释：
```python
# 计算含税总价（增值税 13%）
c = a + b * 0.13
```
